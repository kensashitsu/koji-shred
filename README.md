# ズタッ！

画面をゆびで**ゴシゴシ擦って温度を上げ**、ゆらゆら動く「適温の帯」に合わせてデンプンを糖に分解する、発酵をテーマにした爽快ブラウザゲームです。単一の HTML ファイルのみ・外部ライブラリなしで動きます。

## ▶ 遊ぶ

**https://kensashitsu.github.io/koji-shred/**

スマートフォン（iPhone など）のブラウザでもそのまま遊べます。

## あそびかた

- 画面を**速く往復**して擦るほど温まります（片道スワイプより往復が効きます）
- 温度計が**適温の帯**にあるとき擦ると、でんぷんが糖に分解 → **全部こわせばクリア**
- **熱くしすぎると酵素が失活**（3回で終了）。アツくなったら手を止めて冷ます
- ただし**冷やしすぎもNG**。54℃を下回ると酸敗ゲージが上昇し、満タンでゲームオーバー
